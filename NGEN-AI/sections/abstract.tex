\begin{abstract}
Retrieval-Augmented Generation (RAG)~\cite{lewis2020retrieval} has become the dominant paradigm for grounding large language model outputs in external knowledge, yet practitioners face a vast configuration space---chunking, embedding, retrieval, reranking, prompting, generation model---with limited guidance on how components interact. We present \toolname{}, a modular benchmarking framework that attaches to existing RAG pipelines as a non-invasive evaluation layer, exploring the configuration space via cartesian-product sweeps with MLflow tracking end-to-end. Evaluation combines LLM-based quality (RAGAS), IR measures (nDCG, Hit@k, Recall@k), NLG metrics (BERTScore, METEOR, ROUGE-L, BLEU), and runtime indicators (latency, throughput). We validate the framework on 182 configurations across seven models on SQuAD, revealing non-obvious interactions: MMR retrieval improves nDCG@5 by 8--9pp and Recall@5 by up to 31pp over similarity search; detailed prompt templates raise faithfulness by up to 12pp for capable models; and larger generation models trade latency for answer quality. The framework enables systematic, comparable evaluation of RAG components with minimal integration effort.

\end{abstract}
