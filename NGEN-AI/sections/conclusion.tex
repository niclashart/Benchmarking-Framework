\section{Conclusion}
\label{sec:conclusion}

We presented \toolname{}, a non-invasive, modular benchmarking framework for systematic RAG configuration analysis. Across 182 SQuAD configurations, prompt specificity was the strongest lever for faithfulness, MMR improved retrieval coverage while sometimes reducing grounding, and chunk-size effects were secondary to prompt and retrieval choices. Multi-metric evaluation is necessary: lexical metrics favor concise SQuAD-style answers, whereas faithfulness and BERTScore reward context-grounded semantic content.

Runtime measurements from the independent LLM performance harness show that local deployment can be competitive for batch evaluation, especially with Qwen3.6-27B-NVFP4 on consumer hardware. Future work will extend evaluation to additional datasets, reranking, HyDE query expansion, embedding-model comparisons, and larger samples.
